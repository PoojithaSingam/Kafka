## Prerequisities
JDK8
Zookeeper
Kafka
Maven

## Getting Started
- Open H07 folder on your system and execute the following commands

# Start Zookeeper
- .\01start_zookeeper.bat

# start Kafks
- .\02start_kafka.bat

# Install Jdk8
- Open command prompt as administrator
-------------
choco install jdk8

# Install Maven
- Open command prompt as administrator
------------
choco install maven
